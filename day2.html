WEB DEVELOPMENT TASK

Build a Multi-Section Blog Page

Objective: Create a multi-section blog page using HTML to demonstrate your web development skills, including formatting content, creating links, embedding media, and organizing information.

TASK REQUIREMENTS
  • Follow HTML5 structure: <!DOCTYPE html>, <html>, <head>, <body>
  • Use heading tags, paragraphs, bold, italic, marquee, subscript, superscript, button, image, table, lists.
  • Add internal, external, and downloadable links.
  • Embed video, audio, and iframe tags.
  • Create sections like welcome, topics, media, and footer.
INSTRUCTIONS
1 Set up the HTML5 structure.
2 Add headings, paragraphs, formatting tags and buttons.
3 Create lists and links.
4 Add image, table, video, audio and iframe.
SUBMISSION CHECKLIST
  • ✅ Proper HTML5 structure
  • ✅ Headings and paragraphs
  • ✅ Lists and links
  • ✅ Table and image
  • ✅ Video, audio and iframe
